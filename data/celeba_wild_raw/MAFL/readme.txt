Multi-Attribute Facial Landmark (MAFL) dataset contains 20,000 face images which are annotated with (1) five facial landmarks, (2) 40 facial attributes. The images and annotations are included in the CelebA dataset (http://mmlab.ie.cuhk.edu.hk/projects/CelebA.html).

The files "training.txt" and "testing.txt" include the according file list.
